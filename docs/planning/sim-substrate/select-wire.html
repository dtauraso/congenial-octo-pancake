Select (wire) — motion view

A goroutine reaches a select. It extends a feeler to every case at once, in parallel — no case is checked first or last; the wires are simultaneous. Each feeler lights up if its case is ready. If at least one feeler is lit, exactly one is chosen (the rule: lowest-index lit; the rebuild plan picks its own determinism story), the goroutine commits along that wire, and the unchosen feelers retract. If no feeler is lit and no default exists, the goroutine parks at the select fan-out and waits.

What this view emphasises: the parallel reach, the moment of commit, and the asymmetry between the lit-and-chosen wire and the lit-but-not-chosen wires that retract. Static readiness flags belong to the snapshot view.
