Go Runtime — nodes/ + cmd/ + main.go
main.go
Entry point: RunTest wires topology, runs for duration, dumps Trace JSON-lines to .probe/
cmd/topogen/main.go
topogen CLI: reads topology.json, emits nodes/Wiring/wiring.go with safe identifier validation
cmd/topogend/main.go
topogend daemon: file-watch wrapper around topogen, re-runs on topology.json change
nodes/Wiring/Wiring.go
Generated Wire() function: allocates Go channels and constructs the full node graph
nodes/Line/Line.go
Hand-wired reference topology (Line circuit); channel map encoding the node graph
nodes/SafeWorker/SafeWorker.go
Base goroutine wrapper: context cancellation, panic recovery, Trace hooks
nodes/InputNode/InputNode.go
Boundary node: drains an input channel and forwards values downstream one at a time
nodes/AndGateNode/AndGateNode.go
AND gate: buffers ports a+b; emits out=1 iff both==1, then clears
nodes/ReadGateNode/ReadGateNode.go
ReadGate: fires only when all input slots filled; passes through slots[0] value
nodes/ReadLatchNode/ReadLatchNode.go
ReadLatch: buffers on "in"; emits out+ack on "release"; holds until released
nodes/PartitionNode/PartitionNode.go
Partition: state machine NotInitialized→Growing→Stopped driven by value=1 on "in"
nodes/EdgeNode/EdgeNode.go
Edge detector: buffers left+right; emits XOR on three outputs; contrast detection primitive
nodes/InhibitorNode/InhibitorNode.go
Inhibitor: blocks propagation when inhibit signal present; lateral inhibition primitive
nodes/ChainInhibitorNode/ChainInhibitorNode.go
Chain inhibitor: passes signal unless inhibited; forms inhibitor chains across node sequences
nodes/EdgeInhibitorNode/EdgeInhibitorNode.go
Edge-scoped inhibitor: inhibition gated by edge detection signal
nodes/InhibitRightGateNode/InhibitRightGateNode.go
Inhibit-right gate: passes left input; suppresses right when inhibit active
nodes/SyncGateNode/SyncGateNode.go
Sync gate: buffers a+b; emits release=1 when both arrive (same shape as AND but different semantics)
nodes/StreakDetector/StreakDetector.go
Streak detector: buffers old+new; emits streak=1 when signs match, done=1 always
nodes/StreakBreakDetector/StreakBreakDetector.go
Streak-break detector: buffers old+new; emits done=1 when signs differ
nodes/TransferInhibitorNode/TransferInhibitorNode.go
Transfer inhibitor: passes value through while transferring inhibition to downstream
Trace/Trace.go
Event recorder: captures (node, port, value, time) tuples; serialises to JSON-lines
Trace/Resolve.go
Converts raw (node,port) event keys to canonical edge-keyed form using spec
